🔧 What is SmartHire AI?
SmartHire AI is a Streamlit dashboard that makes the hiring process faster and easier.
It automatically reads resumes, gives candidates a score, ranks them, and even supports email automation.

🚀 Features
Resume Parsing → Extracts details from candidate resumes.

Skill Extraction → Finds important skills from the resume.

Scoring System → Gives each candidate a score based on skills and experience.

Ranking System → Ranks all candidates for easy comparison.

Email Automation → Lets HR send and receive emails directly.

Streamlit Dashboard → A clean, interactive interface where everything is in one place.
